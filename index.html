What is it?

This is a component javascript (without Jquery library to make your project more less) to put a simple slide show on your web site.

When we need to show tip code on web, many peoples uses Gifs or video, but these ways are not usefefull for some little tricks.

Wiht slider SBS you can put your slides (images) on sequencials and put control to your user to teach some thing. It's easy and light to uses.

How to Use

  • Press ARROW left/right to change slide
  • Press ARROW KEYBOARD left/right to change slide without use mouse
  • Get code - https://github.com/renato-js/step-by-step.
